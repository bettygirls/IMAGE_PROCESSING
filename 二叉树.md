# 二叉树

  二叉树度为0的结点数为度为2的结点加1
  n0 = n2 + 1

  完全二叉树总结点n 叶子节点有几个：
  n0 = n2 + 1
  n = n0 + n1 + n2
  
  完全二叉树度为1的只有一个或0个

  所以 n0 = (n + 1) /2
       或者 n0 = n / 2  选择计算出来是整数的

# 继承  默认的继承方式是私有继承

## 多继承
- 多继承即一个子类可以有多个父类，它继承了多个父类的特性
```
class A : public B, public C{
};
```
## 多重继承
- 是一个类可以同时继承多个父类的行为和特征功能
```
class B{
};
class C : public B{
};
class A : public C{
};

```
  
# 多态
- 重载来实现，运行的时候确定，是动态联编。
- 在基类的函数前加上virtual关键字，在派生类中重写该函数，运行时会根据对象的实际类型来调用相应的函数。
  如果对象类型是派生类就调用派生类的函数，如果对象类型是基类就调用基类的函数
